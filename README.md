# Multimodal-AI-for-Phising-Website-detection
This project proposes the development of a Multimodal Artificial Intelligence System for  Phishing Website Detection, which integrates multiple data sources such as textual,  structural, and visual features to enhance detection accuracy and reliability. 
# Introduction : 
With the rapid expansion of digital communication and online services, cybersecurity threats 
have become increasingly significant. Among these threats, phishing websites represent one 
of the most common and harmful forms of cyberattacks, aiming to steal sensitive user 
information such as login credentials, financial data, and personal details. Traditional 
detection techniques often rely on single-source analysis, such as URL features, which may 
not be sufficient against sophisticated phishing strategies. 

This project proposes the development of a Multimodal Artificial Intelligence System for 
Phishing Website Detection, which integrates multiple data sources—such as textual, 
structural, and visual features—to enhance detection accuracy and reliability. The study aims 
to explore advanced AI techniques to build a more robust and effective phishing detection 
framework.
